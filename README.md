## 🇮🇳 What is ArziWala?

**ArziWala** (अर्ज़ीवाला) is a free, secure, and instant letter generator designed specifically for Indian users. It helps you create professional official letters — bank applications, police FIRs, office requests — without any hassle, login, or cost.

> **अर्ज़ी** (Arzi) = Application/Petition in Hindi & Urdu

Whether you're a student, professional, or senior citizen — if you've ever struggled to draft a formal letter, ArziWala is built for you.

## 🚀 How It Works

```
Step 1 → Choose Template
         Select what you need - ATM Lost, Cheque Book, FIR, etc.

Step 2 → Fill Details
         Enter simple details like Name, Account No, Date, etc.

Step 3 → Download / Print
         Get a professional PDF ready to print instantly.
```

**That's it. Three steps. Done.** ✅

## 🏁 Getting Started

### Prerequisites
- Any modern web browser (Chrome, Firefox, Safari, Edge)
- No installation required for end users

### For Developers

```bash
# Clone the repository
git clone https://github.com/yourusername/arziwala.git

# Navigate to the project directory
cd arziwala

# Open in browser
open index.html
# OR use a local server
npx serve .
# OR
python -m http.server 8000
```

### Using npm (if applicable)

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
